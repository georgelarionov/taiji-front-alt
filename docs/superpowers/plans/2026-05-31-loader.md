# Лоудер сайта — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Декоративный брендовый сплэш-лоудер (~4с): плавно проявляет элементы, одновременно «рисует» каллиграфический круг, держит паузу и уезжает вверх; показывается один раз за сессию.

**Architecture:** Разметка `Loader.astro` рендерится во всех страницах через `BaseLayout`, но видна только при классе `html.loading`, который инлайн-`is:inline` скрипт в `<head>` ставит ДО пейнта — лишь при первом заходе за сессию и без `prefers-reduced-motion`. Анимацию ведёт `src/lib/loader.ts` (GSAP-таймлайн) по тому же lifecycle-контракту, что `smooth.ts` (`astro:page-load` / `astro:before-swap`). Круг рисуется через SVG-`<mask>` с анимацией `stroke-dashoffset` (залитую тонкую линию нельзя чертить dashoffset напрямую). На выходе диспатчится `taiji:loader-done` для будущей состыковки с hero (hero в этой работе НЕ трогаем).

**Tech Stack:** Astro 6, Tailwind 4 (utility-классы + токены), GSAP 3.15 (`gsap` + free `gsap/CustomEase`), inline SVG mask.

```ts
// Рантайм лоудера: GSAP-таймлайн (fade-in элементов + «отрисовка» круга через
// маску, затем уход вверх). Лоудер виден только при html.loading (ставит
// инлайн-скрипт в BaseLayout до пейнта: первый заход за сессию, без reduced-motion).
//
// Lifecycle как у smooth.ts: при <ClientRouter /> скрипты не перезапускаются на
// навигации, поэтому init на astro:page-load, teardown на astro:before-swap.

import { gsap } from "gsap";
import { CustomEase } from "gsap/CustomEase";

gsap.registerPlugin(CustomEase);
// Ease ухода — из референса пользователя.
CustomEase.create("loader", "0.65, 0.01, 0.05, 0.99");

// Тайминги (сек) — подгоняй здесь.
const FADE_IN = 0.8; // длительность fade-in элементов
const DRAW = 2.4; // длительность отрисовки круга (одновременно с fade-in)
const HOLD_UNTIL = 3.2; // момент начала ухода (после отрисовки — пауза)
const EXIT = 0.9; // длительность ухода вверх

const SESSION_KEY = "taiji:loaded";

let tl: gsap.core.Timeline | null = null;

function finish(wrap: HTMLElement) {
  sessionStorage.setItem(SESSION_KEY, "1");
  document.documentElement.classList.remove("loading"); // разблок скролла
  wrap.style.display = "none";
  document.dispatchEvent(new CustomEvent("taiji:loader-done"));
}

function run() {
  const root = document.documentElement;
  // Гейт. Класс ставит инлайн-скрипт только на первом заходе за сессию без
  // reduced-motion → отсутствие класса = SPA-переход / повтор / reduced-motion / no-JS.
  if (!root.classList.contains("loading")) return;

  const wrap = document.querySelector<HTMLElement>("[data-loader]");
  if (!wrap) {
    root.classList.remove("loading");
    return;
  }

  // Доп. страховка от reduced-motion: мгновенно завершить, без анимации.
  if (window.matchMedia("(prefers-reduced-motion: reduce)").matches) {
    finish(wrap);
    return;
  }

  const fades = gsap.utils.toArray<HTMLElement>("[data-loader-fade]");
  const ring = wrap.querySelector<SVGCircleElement>("[data-loader-ring]");

  tl = gsap.timeline({
    defaults: { ease: "power2.out" },
    onComplete: () => finish(wrap),
  });

  // Плавное проявление всех элементов (кроме круга).
  tl.fromTo(
    fades,
    { autoAlpha: 0, y: 12 },
    { autoAlpha: 1, y: 0, duration: FADE_IN, stagger: 0.06 },
    0,
  );

  // Одновременно — «отрисовка» круга: маска dashoffset 100 → 0.
  if (ring) {
    tl.fromTo(
      ring,
      { strokeDashoffset: 100 },
      { strokeDashoffset: 0, duration: DRAW, ease: "power1.inOut" },
      0,
    );
  }

  // Уход вверх (bottom→top) после паузы.
  tl.to(wrap, { yPercent: -100, duration: EXIT, ease: "loader" }, HOLD_UNTIL);
}

function cleanup() {
  tl?.kill();
  tl = null;
}

document.addEventListener("astro:page-load", run);
document.addEventListener("astro:before-swap", cleanup);
```
